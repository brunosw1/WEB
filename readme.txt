EMPEZAMOS:
1.descomprimir archivo zip en el escritorio(recomendado) o bien donde usted prefiera
2.abrir la cmd de windows o bien si esta en otro Sistema Operativo(ej. Linux, Mac) abrir su respectivo intérprete de comandos
3.dirigirse por medio de su intérprete de comandos a la ubicacion donde tiene la carpeta descomprimida del proyecto(ej. en cmd seria "cd (destino)" hasta encontrar la carpeta)
4.ya en la carpeta del proyecto en este caso "WEB", activar un entorno virtual(ej. en cmd seria "entorno\Scripts\activate")
5.ya en el entorno virtual hacer el sig comando "pip install -r requirements.txt", asi instalara lo necesario para usar el proyecto sin afectar su sistema
6.ahora dirigirse a la carpeta proyecto(ej. en cmd seria "cd proyecto")
7.lo siguiente seria levantar el servidor de Django escribiendo lo sig. "python manage.py runserver"
8.el servidor estara funcionando y listo para visualizar en un navegador con su url proporcionado por defecto (http://127.0.0.1:8000/)
